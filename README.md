# ¿Qué se cuece en Rusia?

## Descripción:
Análisis de las palabras más frecuentes en los titulares del periódico ruso "The Moscow Times".

## Propósito:
Conocer de manera rápida y visual lo que más revuelo tiene en Rusia en estos momentos.
